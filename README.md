# Visitors_data_Analysis

# 📊  Visitors Data Analysis

## 📌 Overview

This project focuses on analyzing visitor data for Midbrains Institute using Power BI. The dashboard provides insights into student profiles, visit purposes, qualifications, and overall visitor trends. It helps stakeholders understand user behavior and make data-driven decisions.

---

## 📂 Dataset

The dataset contains information about visitors, including:

* Name
* Email
* Qualification
* Profile/Interest (e.g., Web Developer, Data Analyst)
* Purpose of Visit (Job, Internship, etc.)
* Visit Date

---

## 🛠️ Tools & Technologies

* Python (Pandas) – Data cleaning and preprocessing
* SQL Server (SSMS) – Data storage and querying
* Power BI – Data visualization and dashboard creation

---

## 🧹 Data Cleaning Steps

* Removed missing and null values
* Standardized column names (lowercase & underscores)
* Converted date columns into proper format
* Removed duplicates
* Handled inconsistent category values

---

## 📈 Key Insights

* Majority of visitors are Bachelor’s students
* Most users are interested in Web Development and Java Development
* A large portion of visitors are job seekers
* Some data fields had missing “purpose” values
